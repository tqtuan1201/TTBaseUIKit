# Pattern: Header

## Intent
- Provide title and optional actions

## UIKit Building Blocks
- `TTBaseHeaderView`
- `TTBaseUILabel` with `TYPE.HEADER` or `TYPE.HEADER_SUPER`

## SwiftUI Building Blocks
- `TTBaseSUIText` with `.HEADER` or `.HEADER_SUPER`

## Rules
- Use `TTFont.HEADER_H` or `TTFont.HEADER_SUPER_H`
- Use `TTView.textHeaderColor`
