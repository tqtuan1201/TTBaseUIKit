# Pattern: Avatar

## Intent
- Circular image or icon with consistent sizing

## UIKit Building Blocks
- `TTBaseIconCircleView`
- `TTUIImageCircleView`
- `TTBaseUIImageView` with `CircleDrawer`

## SwiftUI Building Blocks
- `TTBaseSUICircleImage`

## Rules
- Use `TTSize.H_ICON` or `TTSize.H_SMALL_ICON` sizes
- Use `drawCircle()` for UIKit views
