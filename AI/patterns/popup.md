# Pattern: Popup

## Intent
- Present content modally with custom transitions

## UIKit Building Blocks
- `TTBasePopupViewController`
- `TTPopupViewController`
- `TTCoverVerticalPresentationController`

## SwiftUI Building Blocks
- `TTBaseHostingController` to host SwiftUI views

## Rules
- Use `PRESENTATION_TYPE` from `TTBaseNotificationViewConfig` when relevant
