# Pattern: Notification

## Intent
- Toast-like UI for feedback

## UIKit Building Blocks
- `TTBaseNotificationView`
- `TTBaseNotificationViewConfig`

## SwiftUI Building Blocks
- `TTBaseNewVersionPopupView` for versioning notifications

## Rules
- Use `TTView.notificationBgSuccess`, `TTView.notificationBgWarning`, `TTView.notificationBgError`
- Position using `TTBaseNotificationViewConfig.POSITION`
