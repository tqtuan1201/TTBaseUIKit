# Pattern: Grid

## Intent
- Show tiled content with consistent spacing

## UIKit Building Blocks
- `TTBaseUICollectionView`
- `TTBaseUICollectionViewCell`
- `TTBaseShadowCollectionViewCell`

## SwiftUI Building Blocks
- `TTBaseSUILazyVGrid`
- `TTBaseSUILazyHGrid`

## Rules
- Use `TTSize.P_S` or `TTSize.P_M` for spacing
- Use `TTSize.H_CELL_COLECT` for cell height
