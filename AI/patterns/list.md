# Pattern: List

## Intent
- Provide consistent row styling and separators

## UIKit Building Blocks
- `TTBaseUITableView`
- `TTBaseUITableViewCell`
- `TTBaseUITableViewHeaderFooterView`

## SwiftUI Building Blocks
- `TTBaseSUIList`
- `TTBaseSUIVStack`

## Rules
- Use `TTView.tableRow*` colors
- Use `TTSize.H_CELL` for default row height
