# Pattern: Form

## Intent
- Consistent field rows and validation feedback

## UIKit Building Blocks
- `TTLabelTextFieldView`
- `TTIconLabelTextFieldView`
- `TextViewWithPlaceHolderView`
- `TTBaseUITextField`

## SwiftUI Building Blocks
- `TTBaseSUITextField`
- `TTBaseSUIText`

## Rules
- Use `TTSize.H_TEXTFIELD` for field height
- Use `TTView.lineDefColor` for separators
