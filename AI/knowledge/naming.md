# Naming Conventions

## Prefixes
- `TTBase*` for UIKit base components
- `TTBaseSUI*` for SwiftUI base components
- `*Config` for tokens

## Tokens
- `TTSize`, `TTView`, `TTFont`, `TTStyle`, `TTParam`

## File Placement
- UIKit base: `CustomView/BaseUIView/**`
- UIKit controllers: `CustomView/BaseUIView/ViewController/**`
- SwiftUI base: `SwiftUIView/BaseViews/**`
- SwiftUI modifiers: `SwiftUIView/Extension+Configs/**`
